# IEC-61499-Image-maker

Программа для автоматического создания изображения функционального блока *IEC-61499* на основе его структуры, описанной в *XML* формате.

**Локальная часть** — генерация схемы функционального блока (*PNG* и *SVG*) из файла формата *.fbt*.

**Онлайн часть** — *Telegram*-бот, принимающий от пользователя *.fbt* файл с описанием функционального блока и возвращающий автоматически сгенерированные изображения.

**МАИ 307**
